Array Operations

AppleAccelerate wraps Apple's vecLib (vv* and vDSP_* functions) to provide accelerated element-wise operations on Array{Float32} and Array{Float64}.

These functions are not exported to avoid conflicts with Base. Access them via the AppleAccelerate. prefix or use @replaceBase to override Base functions.

Broadcasting

AppleAccelerate overrides Base.copy and Base.copyto! for Broadcasted objects, so that broadcasting syntax like f.(X) automatically uses the accelerated implementation.

Replacing Base Functions

AppleAccelerate.@replaceBase — Macro
@replaceBase(f1, f2, ...)

Replace Base functions with their AppleAccelerate equivalents for Array{Float32} and Array{Float64}. After calling this macro, expressions like sin.(X) will use the accelerated Accelerate implementation transparently.

Supported functions: ceil, floor, sqrt, exp, log, sin, cos, tan, asin, acos, atan, sinh, cosh, tanh, and more. Arithmetic operators +, -, *, / are also supported.

Example

using AppleAccelerate
AppleAccelerate.@replaceBase sin cos exp log
X = randn(Float64, 1000)
sin.(X)  # Now uses Accelerate
